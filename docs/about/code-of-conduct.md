## Goal

RbxPI (Roblox Package Index) is a community project aiming to provide a reliable, open, and maintainable package ecosystem for Roblox / Luau. This code of conduct aims to ensure a **respectful**, **professional**, and **collaborative** environment for everyone involved.

By participating in the RbxPI project, you agree to abide by this code of conduct.

---

## Scope

This code of conduct applies to **all project spaces**, including:

- GitHub repositories (issues, pull requests, discussions, commits)
- RbxPI organization and affiliated projects
- Community spaces (Discord, forums, project-related private discussions)
- Any public or private interaction related to RbxPI

---

## Expected Behaviors

Contributors and community members commit to:

- Being respectful, courteous, and kind toward others
- Accepting constructive criticism and reasoned technical debates
- Communicating in a clear, honest, and professional manner
- Respecting differences in opinions, experiences, and technical levels
- Prioritizing the project's interest and its quality

---

## Unacceptable Behaviors

The following behaviors are strictly prohibited:

- Harassment, intimidation, or threats in any form
- Insults, degrading remarks, or personal attacks
- Discrimination based on gender, origin, religion, disability, sexual orientation, or any other personal characteristic
- Trolling, deliberate provocation, or project sabotage
- Publishing private information without consent
- Plagiarism or appropriation of others' work without attribution

---

## Contribution Standards

In order to maintain a high-quality project:

- Contributions must respect established coding conventions and rules
- Pull requests must be clear, justified, and documented
- Technical discussions must remain factual and reasoned
- Final decisions rest with the project maintainers

Technical disagreements are normal, but must always be handled with respect.

---

## Maintainer Responsibilities

RbxPI maintainers commit to:

- Applying this code of conduct fairly and consistently
- Listening to community feedback and concerns
- Making decisions in the long-term interest of the project
- Intervening promptly in case of inappropriate behavior

---

## Disciplinary Measures

In case of non-compliance with this code of conduct, maintainers may apply, depending on the severity:

- An informal or formal warning
- Removal of inappropriate content
- Locking of issues or discussions
- Temporary or permanent suspension from participating in the project
- Banning from RbxPI-related community spaces

---

## Reporting

Any behavior contrary to this code of conduct can be reported confidentially to the project maintainers.

Reports will be examined seriously and handled with discretion and impartiality.

---

## Attribution

This code of conduct is inspired by open-source project best practices and the principles of the **Contributor Covenant**, adapted to the specific needs of RbxPI.

---

## Acceptance

Any participation in RbxPI implies full and complete acceptance of this code of conduct.

By contributing, you also contribute to maintaining a healthy, open, and sustainable community.
