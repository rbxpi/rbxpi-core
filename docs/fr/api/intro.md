## API Interne

L’API interne de RbxPI regroupe exclusivement les fonctions internes destinées à être appelées et utilisées par les développeurs.

::: warning

Si certaines fonctions présentes dans le code source ne sont pas documentées ici, cela signifie qu’elles ne sont pas prévues pour être manipulées directement.
Elles font partie de l’implémentation interne et peuvent être modifiées ou supprimées sans préavis.

:::
